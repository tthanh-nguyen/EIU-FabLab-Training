# EIU FabLab Training
This repository tracks my 10+ month parallel training and research progress at EIU FabLab.

### 1. STM32 Track
- Learning hardware configuration via **STM32CubeMX** and programming in **Keil C**.
- Working with basic peripherals: GPIOs, Timers, Interrupts, ADC-DMA, and PWM.

### 2. Autonomous Robotics Track (F1TENTH)
- Developing embedded control scripts using **Python** in **Ubuntu Linux**.
- Utilizing **ROS2** basic libraries (`Ackermann`, `Odom`, `LaserScan`) to process sensors and develop an **Automatic Emergency Braking (AEB)** simulation.
